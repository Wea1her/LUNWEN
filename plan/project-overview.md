## 三、论文一：期刊论文（当前主任务）

### 3.1 论文题目

**基于深度强化学习的以太坊交易序列决策排序方法研究**

### 3.2 论文定位

- **类型**：中文期刊 / 学报 / 普通期刊
- **目标篇幅**：5000–8000 字
- **写作形式**：LaTeX
- **研究定位**：方法型 + 仿真实验型论文

### 3.3 核心研究问题

论文核心问题可概括为：

> 在区块 gas 容量、Nonce 依赖和风险约束存在的情况下，将以太坊交易排序建模为逐笔交易选择的序列决策问题，是否能够比 FIFO、Gas 优先和启发式方法实现更优的收益—公平—风险折中。

### 3.4 当前方法概述

当前小论文方法已经明确为：

- 采用 **PPO** 作为强化学习核心算法；
- 将区块构建视为一个 episode；
- 状态由候选交易特征与区块状态组成；
- 动作为“选择某一笔候选交易”或 `STOP`；
- 奖励函数联合考虑：
  - 区块收益；
  - 等待时间公平性；
  - 高风险交易位置惩罚；
  - 非法动作 / 提前停止惩罚。

### 3.5 当前实验协议（V5）

当前实验框架以 V5 方案为准，具有以下特点：

- 主实验、三类鲁棒性与两类消融都使用共享评估池；
- 默认 checkpoint 选模指标为 `two_stage`，即可行率优先、可行子集收益次优先、风险调整收益辅助；
- 强基线正式包含 `Center-Insertion Heuristic` 与 `Dynamic Tri-Objective Greedy`；
- 含参数基线在固定验证池上调参，测试池不参与调参；
- 奖励函数按单步奖励、终止奖励和有效性奖励三层描述；
- 奖励消融采用 `AgeOnly / Age+Risk / Age+TerminalFair / FullBalanced`；
- 结构消融额外报告非法动作率、非法截断率和连续非法动作长度；
- 正式统计以独立训练 seed 为单位，episode 级配对检验仅作诊断；
- 正式报告输出 baseline tuning、method registry、seed-level statistics、推理时间和 V5 protocol manifest。


### 3.6 论文当前三个创新点（建议版）

1. **问题建模创新**  
   将交易排序建模为逐笔交易选择的序列决策过程，而不是在预定义规则之间切换。

2. **目标设计创新**  
   构建融合收益、公平性与风险控制的联合奖励机制，并通过动作掩码与 STOP 动作增强策略可行性。

3. **实验协议创新**  
   建立统一仿真框架，在主实验、三类鲁棒性、奖励消融、结构消融与行为解释层面系统验证方法有效性。

### 3.7 当前评价指标

#### 主指标

- 区块手续费收益；
- 等待时间均衡性（Jain 指数）；
- 风险暴露度；
- Gas 利用率；
- 风险交易平均相对位置；
- 打包比例。

#### 辅助指标

- Top-10% Risk Ratio；
- Late-Arrival Promotion Rate。

### 3.8 当前基线方法

当前正式基线包括 7 类：

1. FIFO
2. Gas Priority
3. Heuristic Risk-Aware
4. Fee-Risk Linear
5. Fair-Fee Greedy
6. Center-Insertion Heuristic
7. Dynamic Tri-Objective Greedy

其中后两类为 V5 强启发式基线。`center_aware` 仅作为 legacy alias 保留，不再作为论文正式方法名。


### 3.9 当前主要风险点

目前最需要持续关注的不是“能不能跑”，而是以下三类风险：

1. **实验结果风险**：可行率提升与收益回补是否能在多 seed 下同时稳定；
2. **统计风险**：探索性结果被误写为正式显著性结论；
3. **叙事风险**：若只给单一 ranking，容易与多目标结论冲突。

## 四、论文二：硕士毕业论文（中长期主线）

### 4.1 预定题目

**基于约束深度强化学习的以太坊 DEX 交易排序与风险防护研究**

### 4.2 与期刊论文的关系

毕业论文将以当前期刊论文为基础，向更复杂场景扩展：

- 从通用候选池排序扩展到 DEX / AMM 场景；
- 从多目标奖励优化扩展到约束强化学习；
- 从轻量仿真扩展到多场景、更复杂的实验设定；
- 从单一方法验证扩展到完整方法体系。

### 4.3 可能扩展方向

- 动态风险约束与拉格朗日乘子调节；
- DEX 交易依赖与滑点约束；
- 多智能体排序博弈；
- PBS / FSS / Batch Auction 等机制对比；
- 更接近真实链上数据的实验评估。
