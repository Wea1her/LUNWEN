## 论文一：期刊论文（小论文）

- **题目**：基于深度强化学习的以太坊交易序列决策排序方法研究
- **类型**：中文学报 / 普通期刊
- **篇幅**：5000-8000字
- **输出格式**：LaTeX
- **核心问题**：把以太坊交易排序建模为"逐笔交易选择"的序列决策过程后，是否能比 FIFO、Gas 优先和启发式方法取得更好的综合效果

### 三个创新点

1. 将交易排序建模为面向区块构建的逐笔交易选择过程（而非在少量预定义排序规则之间切换）
2. 设计兼顾收益、公平性与风险抑制的简化奖励机制
3. 构建可配置的通用交易排序仿真环境，支持候选交易池、区块 gas 限制、风险标签和多种基线排序

### 研究内容

- 问题建模：episode = 一个区块构建过程，状态 = (候选交易集合, 剩余gas, 风险特征)，动作 = 选下一笔交易或STOP
- 排序环境：轻量仿真环境（非 Geth 私链）
- 方法实现：基础 DRL 排序器（PPO 或简化 DQN）
- 基线对比：FIFO、Gas Priority、Heuristic Risk-Aware Sort

### 实验目标

- 逐笔序列决策排序是否优于传统规则
- 风险类指标是否下降，公平类指标是否提升
- 在不同风险负载下是否有基本鲁棒性
